# 部署文档 - 生产环境上线指南

## 一、服务器环境要求

- 操作系统：Linux（Ubuntu 20.04+ / CentOS 7+）
- Python：3.11+
- Node.js：18+
- Web服务器：Nginx
- 应用服务器：Gunicorn
- Chromium（用于 SSG 预渲染）

---

## 二、部署流程

### 1. 后端部署（Django + Gunicorn + Nginx）

```bash
cd backend
pip install -r requirements.txt
pip install gunicorn
python manage.py migrate
python manage.py collectstatic --noinput
python manage.py shell < api/init_data.py
```

启动 Gunicorn：
```bash
gunicorn config.wsgi:application -b 127.0.0.1:8000 --workers 4 --daemon
```

### 2. 前端构建 + SSG 预渲染

```bash
cd frontend
pnpm install
pnpm run build:prerender    # 构建 + 预渲染所有路由
```

构建产物位于 `frontend/dist/`，包含：
- 所有路由的预渲染 HTML（含真实正文 + JSON-LD）
- 哈希后的 JS/CSS 资源
- `robots.txt`、`sitemap.xml`、`llms.txt`

### 3. Nginx 配置（同域部署，推荐）

```nginx
server {
    listen 80;
    server_name laborsaving-arm.cn www.laborsaving-arm.cn;

    # 前端预渲染静态文件
    root /path/to/frontend/dist;
    index index.html;

    # SEO 静态文件
    location = /robots.txt { }
    location = /sitemap.xml { }
    location = /llms.txt { }

    # API 代理到 Django
    location /api/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # Django Admin
    location /admin/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # Django 静态文件
    location /static/ {
        alias /path/to/backend/staticfiles/;
    }

    # 前端 SPA fallback（预渲染的 HTML 优先）
    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

## 五、运维监控

- 数据库定期备份：`cp backend/db.sqlite3 db_backup_$(date +%Y%m%d).sqlite3`
- 日志查看：`tail -f /var/log/nginx/access.log`
- 进程监控：Supervisor 守护 Gunicorn 进程
- 统计分析：登录百度统计后台查看流量数据
