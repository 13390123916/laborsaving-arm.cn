## 一、本期目标与约束

**目标**：在「不破坏既有 SEO/GEO 收录架构」前提下，将国内官网前端整体升级为**深色工业风 + 品牌红（#f02b23）强调**视觉体系，提升专业感与移动端阅读体验，对齐国际站设计语言。

**硬性约束（不可妥协）**：
1. 不改动预渲染脚本、vite 配置、路由、seo 插件、`index.html` 结构 —— 收录架构完整保留。
2. 不改任何 `<template>` 业务结构与 `<script>` 逻辑（尤其 `setTDK`、JSON-LD 注入、表单转化埋点）。
3. 仅改动全局 `style.css`、全局 `App.vue` 布局样式、各视图 `<style scoped>` 样式块。
4. 全站移动端优先，无横向溢出、无布局错乱。

## 三、Phase 3：各页面组件改版

仅替换各视图 `<style scoped>` 样式块，`<template>` 与 `<script>` 逻辑（含 SEO/JSON-LD/埋点）零改动。

| 视图 | 改版要点 |

| `News.vue` | 分类 tab 迁入深色面；封面占位图由纯红改为深色渐变 + 红字；分类标签改为红浅底；文章卡 hover 红边浮起 |
| `NewsDetail.vue` | 分类标签红浅底；点赞按钮 hover/激活态红；相关推荐卡迁入深色面 + hover 红边；封面图加描边 |
| `Faq.vue` | 问答卡迁入深色面；Q 标记红圆；答案短句红；箭头展开态转红；分类 tab 深色态 |
| `Contact.vue` | 联系信息标题转白、勾选项转红；表单输入框迁入深色输入面（`--bg-input`），聚焦红色描边 + 红浅光晕；成功绿/失败红提示 |

### 3.1 一致性规则
- 所有 CTA 按钮统一 `.btn`（红底白字）；次级动作使用 `.btn-ghost`（透明 + 强描边）。
- 区块标题统一红色短下划线（`::after` 52px×3px）。
- 卡片 hover 统一「红边 + 微上浮 2~3px」。
- 激活态 / 关键标识统一使用 `--primary`（红），避免多色杂乱。

# 生产构建（含 SSG 预渲染，需 Django 后端在 :8000 + Chromium）
